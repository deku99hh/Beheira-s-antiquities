# جولة في متحف البحيرة الافقي (Beheira Museum Tour)


موقع ويب تعريفي سياحي يهدف إلى تسليط الضوء على المعالم الأثرية والسياحية في **محافظة البحيرة** بجمهورية مصر العربية. يأخذ الموقع الزائر في جولة افتراضية بين المباني التاريخية، التماثيل، والأماكن المقدسة، مع توفير نبذة تاريخية وصور لكل معلم.

## مميزات المشروع

* **تصميم متجاوب (Responsive Design):** يعمل بكفاءة على الهواتف المحمولة وأجهزة الكمبيوتر.
* **تصنيف المعالم:** تقسيم المحتوى إلى أقسام واضحة (الأماكن، التماثيل، المباني) لسهولة التصفح.
* **صفحات تفصيلية:** يحتوي الموقع على صفحات فرعية لكل معلم أثري (مثل: أوبرا دمنهور، حجر رشيد، أديرة وادي النطرون) لعرض المزيد من التفاصيل.
* **واجهة جذابة:** استخدام صور عالية الجودة وتنسيقات CSS حديثة تعكس الطابع التراثي.
* **سهولة التنقل:** قائمة تنقل علوية (Navbar) ذكية وزر للعودة للأعلى (Scroll to Top).
